\subsection{Type de fichier}
\label{sec:file}

\paragraph{}Pour ce projet nous avions le choix entre 2 types de fichiers pour le stockage de la base de données:
\begin{itemize}
\item Extensible Markup Language (XML)
\item Comma-separated values (CSV)
\end{itemize}

\paragraph{}Nous avons choisi d'utiliser un fichier CSV pour stocker notre jeu de données, et ainsi réutiliser les connaissances accquises en cours.

\subsection{Charte graphique}
\paragraph{}Notre site web se veut sobre, élégant et intuitif, c'est pour quoi nous avons utilisé une palette de couleur assez sobre et pas trop vive. Le bleu est très présent sur notre site web, en effet c'est une couleur sobre, ministériel, qui instaure un sérieux, ce qui créé une cohérence avec l'objet du site web.
