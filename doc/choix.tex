\subsection{Type de fichier}
\label{sec:file}

\paragraph{}Pour ce projet nous avions le choix entre 2 types de fichiers pour le stockage de la base de données:
\begin{itemize}
\item Extensible Markup Language (XML)
\item Comma-separated values (CSV)
\end{itemize}

\paragraph{}Nous avons choisi d'utiliser un fichier CSV pour stocker notre jeu de données, et d'ainsi réutiliser les connaissances accquises en cours.
